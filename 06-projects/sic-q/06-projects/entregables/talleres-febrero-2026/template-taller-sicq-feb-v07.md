# TEMPLATE COMUNIDAD v.07

## Para diseño de rotafolio (tamaño A1 o equivalente)

---

# COMUNIDAD

**MESA #____**

---

## Preguntas principales (arriba, destacadas):

### ¿A quiénes necesitamos en la mesa?
### ¿Qué podrían lograr juntos que hoy ninguno puede lograr solo?

**→ Un sistema de inteligencia colectiva se diseña colectivamente. ¿Quiénes faltan en esta conversación?**

---

## Área principal — Red de conexiones:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│                                                                     │
│                                                                     │
│                                                                     │
│                            SIC-Q                                    │
│                                                                     │
│                                                                     │
│                     /  |  |  |  |  |  \                             │
│                                                                     │
│                                                                     │
│  • Peguen los nodos (sectores/instituciones/comunidades)            │
│    alrededor del SIC-Q                                              │
│                                                                     │
│  • En cada nodo escriban:                                           │
│    ¿Qué sabe este sector que los demás necesitan?                   │
│    ¿Qué necesita que otros podrían darle?                           │
│                                                                     │
│  • Conecten nodos entre sí cuando puedan lograr algo juntos         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

---

## Instrucciones al pie:

**PASOS:**

1. **Escriban nodos** en post-its: sectores, instituciones, comunidades, organizaciones
2. **Peguen alrededor del SIC-Q** (que está en el centro)
3. **En cada nodo escriban:**
   - **Sabe:** ¿qué sabe este sector que los demás necesitan?
   - **Necesita:** ¿qué necesita que otros podrían darle?
4. **Conecten nodos entre sí** cuando dos sectores podrían lograr algo juntos. Anoten qué lograrían.

---

**EJEMPLOS:**

- **Academia → SIC-Q:** Da: metodología, análisis / Recibe: datos reales, problemas concretos
- **Gobierno → SIC-Q:** Da: datos institucionales, procesos / Recibe: diagnósticos, patrones, alertas
- **Industria → SIC-Q:** Da: tendencias de empleo / Recibe: inteligencia del mercado laboral
- **Academia + Industria + SIC-Q:** Juntos: identificar brechas de talento antes de que sean crisis
