## Inteligencia colectiva: qué es y qué no es

| Principio             |                                                                |
| --------------------- | -------------------------------------------------------------- |
| **Agregación**        | Muchas perspectivas independientes → mejor que un solo experto |
| **Diversidad**        | Perspectivas distintas reducen el sesgo                        |
| **Descentralización** | El conocimiento está distribuido, no concentrado               |

---

**No es** votación · **No es** encuesta · **No es** crowdsourcing
Es un sistema que **escucha, agrega, pondera y sintetiza**.

## Lo que cambia cuando le sumas IA

La inteligencia colectiva existe desde hace décadas.
Lo que no existía era la capacidad de operarla **a escala**.

- **Escuchar a miles a la vez** — sin perder ninguna voz
- **Ver lo invisible** — patrones entre fuentes distintas
- **Conversar, no solo recolectar** — diálogos personalizados, cualquier canal, cualquier hora
- **Aprender continuamente** — el sistema mejora con cada interacción

> La inteligencia colectiva es la teoría detrás.
> La IA es lo que permite que funcione a la escala de un estado.

## ¿Cómo funciona un sistema de inteligencia colectiva?

| Capa | |
|---|---|
| **Motor de inteligencia** | IA que cruza datos, identifica patrones, sintetiza — el cerebro del sistema |
| **Herramienta de recopilación** | Captura y estructura información de procesos: consultas, talleres, consejos |
| **Agente conversacional** | Cualquier persona puede preguntar, compartir, obtener información — y recibir respuesta |
| **Canales** | Los medios por donde fluye la información: web, WhatsApp, talleres presenciales |

Lo central es la **inteligencia** — los canales son los medios para alimentarla y distribuirla.

## No es un chatbot — es un sistema de inteligencia

Hay tres cosas distintas que se confunden fácilmente:

| | Qué es | Analogía |
|---|---|---|
| **Un chatbot** | Una interfaz — una ventana donde escribes y recibes texto | El teléfono |
| **Un modelo de IA** | Un motor que procesa lenguaje — como ChatGPT o Claude | El cerebro genérico que sabe de todo pero no conoce tu contexto |
| **Un sistema de IA** | Una arquitectura completa en la nube: datos propios, memoria, procesamiento, canales, reglas, gobernanza | Un equipo completo que conoce tu realidad, aprende, y trabaja para ti 24/7 |

---

**ChatGPT** es un modelo con un chatbot encima. Sabe mucho de todo, pero nada específico de Querétaro. No tiene memoria entre conversaciones. No cruza tus datos. No aprende de tus procesos.

**El SIC-Q** es un **sistema** — lo que hoy se llama IA agéntica: no solo responde preguntas, **actúa**.

| Capacidad | Qué significa |
|---|---|
| **Agentes especializados** | No es un solo bot — son múltiples agentes: uno para planeación, otro para ciudadanía, otro para prospectiva. Cada uno con su contexto y sus datos |
| **Tools (herramientas)** | Los agentes pueden usar herramientas: consultar bases de datos, generar reportes, cruzar fuentes, buscar en documentos — no solo conversar |
| **Skills (habilidades)** | Módulos especializados que se pueden agregar: análisis de sentimiento, georreferenciación, detección de patrones, generación de resúmenes ejecutivos |
| **RAG (Retrieval)** | El sistema busca en los datos reales de Querétaro antes de responder — no inventa, consulta fuentes verificables |
| **Memoria persistente** | Recuerda conversaciones previas, aprende de cada proceso, acumula inteligencia con el tiempo |
| **Multimodal** | Puede procesar texto, voz, imágenes, documentos escaneados — no solo texto escrito |

El chatbot es solo la puerta de entrada. Lo que hay detrás es un sistema de inteligencia con las capacidades más avanzadas que existen hoy.

|---|---|
| **Ciudadanía → Sistema** | Experiencias cotidianas → el sistema agrupa, detecta patrones, genera inteligencia |

La inteligencia crece en cada dirección.
