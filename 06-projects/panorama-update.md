# Panorama Update

Operational intake note for cross-vault state.

Use this note for raw dictation from macOS. HA reads this note, normalizes the input, and distributes relevant updates to the dashboards and project states inside this vault.

## Usage
- Dictate fast; do not self-edit.
- One dated block per update.
- Include wins, blockers, changes in priority, and anything that should propagate inside `ha-ha`.
- If a dictated item should become a draft email or message, HA can move it later to `writing-lab.md` in the appropriate vault.
