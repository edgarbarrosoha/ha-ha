## 1. ¿Qué es la Plataforma?

Una **plataforma de inteligencia colectiva potenciada por IA** que transforma la red de personas egresadas y participantes de Tec Beyond en una comunidad de valor continuo.

**No es solo un CRM.** Es un sistema que:
- Aprende de las interacciones de la comunidad
- Facilita conexiones de negocio entre personas empresarias
- Personaliza contenido y capacitación
- Detecta personas candidatas para futuras generaciones
- Genera insights estratégicos para el Consejo

---

## 2. ¿Cómo Funciona?

### Modelo de Membresía

| Nivel | Incluye |
|-------|---------|
| **Suscripción Base** | Acceso a la comunidad, agentes IA, contenido personalizado |
| **Match de Negocios** | Conexiones facilitadas entre personas empresarias (anonimizadas) |
| **Capacitación Continua** | 2da y 3ra fase de Tec Beyond (lifelong learning) |
| **Experiencias** | Eventos presenciales, rituales, networking |

### Agentes IA

| Agente         | Función                                           |
| -------------- | ------------------------------------------------- |
| **Match**      | Conecta personas con oportunidades de negocio     |
| **Contenidos** | Personaliza capacitación según perfil e intereses |
| **Scout**      | Detecta personas candidatas en redes              |
| **Insights**   | Genera hallazgos para el Consejo                  |

## 3. Arquitectura y Datos

### ¿Quién posee qué?

| Componente                     | Propiedad             | Control                         |
| ------------------------------ | --------------------- | ------------------------------- |
| **Datos de miembros**          | Tec / Consejo León    | 50% Tec                         |
| **Reglas de negocio**          | Tec / Consejo León    | 100% Tec Beyond                 |
| **Arquitectura de plataforma** | Horizons Architecture | Licencia de uso para Tec Beyond |
| **Agentes IA**                 | Horizons Architecture | Licencia para Tec Beyond        |

### Principios de Datos

- Los datos **nunca salen** del control del Consejo
- La infraestructura de datos **puede residir en servidores del Tec** (Azure del Tec o infraestructura propia)
- HA provee la **arquitectura y los agentes**, no almacena datos
- Tec Beyond puede auditar en cualquier momento
- Cumplimiento con privacidad y protección de datos
