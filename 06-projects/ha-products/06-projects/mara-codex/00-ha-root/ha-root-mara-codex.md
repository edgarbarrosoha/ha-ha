# HA Root - Mara Codex

Mara-Codex is the working implementation project for MARA inside `ha-products`.

## North Star

Build the first MARA that is actually usable: a local-first fractal cockpit that can read a markdown workspace, surface what matters now, and help turn central ambiguity into bounded next moves.

## Design Rules

1. Local-first before cloud-first.
2. Markdown-native before platform-dependent.
3. Preview before write.
4. Fractal zoom before generic chat.
5. The interface should feel like thinking, not operating.
6. TaaS alignment matters, but not at the cost of shipping a trustworthy MVP.
