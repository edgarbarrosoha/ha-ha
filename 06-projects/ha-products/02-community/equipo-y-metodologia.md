# Equipo y Metodología de Trabajo

> Extraído de: HA Technology Presentation v2 (Oscar Diaz, CTO — Marzo 2026)

## Ciclo de Desarrollo

| Paso | Quién lidera |
|---|---|
| 1. Especificar | Humano lidera, IA asiste |
| 2. Implementar | IA lidera, humano supervisa |
| 3. Verificar | Híbrido: tests + review |
| 4. Consolidar | Captura de conocimiento |

## Protocolo Interns

| Periodo | Modo |
|---|---|
| Sem 1-4 | Código manual. Sin IA para generar código |
| Sem 5-8 | IA asistida + explicaciones obligatorias |
| Sem 9+ | Augmented coding + día sin IA/semana |

**Razón:** 17% menos comprensión con IA temprana.
